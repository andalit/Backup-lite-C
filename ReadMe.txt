Данная программа предназначена для резервного копирования данных и отправки отчета на электронный адрес.
Была разработана как практичное решение для конкретного предритятия. А так же была представлена автором на защите дипломного проекта в 2015м году в ХНТУ.

Программа распространяется "как есть" по лицензии OpenGPL. Автор не несет отвецтвенности за неправильное использование или использование не по назначению.
